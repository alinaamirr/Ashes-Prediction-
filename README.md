# Ashes Match Outcome & Score Prediction System

## Overview

This project applies Machine Learning techniques to predict Ashes cricket match outcomes and forecast total runs using historical match data. The system analyzes factors such as team form, venue, toss decisions, pitch conditions, and weather to generate predictive insights.

## Features

* Match Outcome Prediction (Home Win, Away Win, Draw)
* Total Runs Forecasting
* Data Cleaning and Preprocessing
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Hyperparameter Tuning using GridSearchCV
* Classification and Regression Model Comparison
* Performance Evaluation with Visualization

## Technologies Used
